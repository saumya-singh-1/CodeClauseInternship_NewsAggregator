# 📰 News Aggregator

A modern news app built with **Next.js + TailwindCSS** that fetches top headlines from the NewsAPI.

## Features
- 🔍 Search for news articles
- 📂 Browse by category (General, Sports, Tech, etc.)
- 🌙 Dark mode UI
- 📱 Fully responsive design

## Tech Stack
- Next.js 14
- Tailwind CSS
- NewsAPI
